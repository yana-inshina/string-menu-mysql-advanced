Консольное меню: строки (substring, регистры, поиск, endsWith) + MySQL + Excel
Требования
JDK 17+
MySQL 8.x (localhost:3306)
Установка БД
В MySQL выполнить:
SOURCE create_db.sql;
При необходимости сменить креды в src/main/java/org/example/Main.java (константы DB_URL / DB_USER / DB_PASS).
Запуск
Сборка JAR:
mvn -q -DskipTests package
Запуск:
java -jar target/string-menu-mysql-advanced-1.0.0-shaded.jar
Проверка пунктов меню
1 — вывести все таблицы из MySQL (проверка подключения к БД и выбранной схемы)
2 — создать/проверить таблицу string_ops
3 — вернуть подстроку по индексам (substring(start, end)), сохранить в MySQL и вывести результат в консоль
4 — перевести строку в верхний и нижний регистры, сохранить в MySQL и вывести в консоль
5 — поиск подстроки и проверка окончания (endsWith) исходной строки; сохранить в MySQL и вывести результат в консоль
6 — экспортировать все записи из string_ops в string_ops.xlsx и вывести путь к файлу в консоль
Для пунктов 3–5 каждая операция сохраняется отдельной строкой в таблицу string_ops (с датой, исходными параметрами и результатами).
Примечания
В п.3 используется стандартная Java-семантика: substring(start, end) — end не включается.
Индексы считаются с нуля. Диапазон проверяется в коде; при ошибке выводится сообщение.
В п.4 сохраняются обе версии — UPPER и lower.
В п.5 сохраняются: искомая подстрока, суффикс для endsWith и булевый результат проверки.
Таблица string_ops создаётся либо через create_db.sql, либо пунктом меню 2.
Файл Excel string_ops.xlsx создаётся рядом с исполняемым JAR (или в корне проекта при запуске из IDE).
При экспорте может выводиться предупреждение вида StatusLogger / log4j2 — на функциональность не влияет.
